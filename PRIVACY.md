# Privacy Policy

**Last updated: March 30, 2026**

## 1. Overview
HorrorStories is a personal automated bot. This privacy policy explains how the application handles data.

## 2. Data We Do NOT Collect
This application does not collect, store, process, or share:
- Personal information from TikTok users
- Viewer data, comments, or interactions
- Any analytics or tracking data

## 3. Data Used to Operate the Bot
The only data used is:
- **TikTok OAuth access token** — stored locally on the operator's machine, used solely to publish videos via the TikTok API. It is never shared with third parties.

## 4. Third-Party APIs
This bot uses the following external services, each governed by their own privacy policies:
- [TikTok API](https://www.tiktok.com/legal/privacy-policy)
- [Anthropic (Claude)](https://www.anthropic.com/privacy)
- [ElevenLabs](https://elevenlabs.io/privacy)
- [Pexels](https://www.pexels.com/privacy-policy/)

## 5. Data Retention
No user data is retained. Generated video files are stored temporarily on the operator's local machine or CI environment and are not distributed beyond TikTok.

## 6. Changes
This policy may be updated at any time. The latest version will always be available in the project repository.

## 7. Contact
For any questions, open an issue in the project repository.
